# Task_APP

iOSのリマインダー的なアプリを作りました。
ローカルでRealmに保存し、バックアップ機能としてFirebaseにアップする必要があります。
#  Requirements

- Swift 4.1

# Configure
CocoaPods
```pod update```

FirebaseからGoogleService-Info.plistを入れてください。
